The bar bodies are painted into a single <canvas> while the axes, grid, zoom, and shared tooltip stay as SVG. Zooming auto-scales the y-axis to the visible bars while keeping the 0 baseline (via zoom.autoScaleYaxis: true), so the bars still read as magnitude. Data labels stay SVG (the hybrid keeps chrome as SVG), so they are turned off here: 10,000 labels would be one DOM node each and unreadable at this density. With that, renderer: 'auto' promotes columns only at 50,000 bars, so this demo selects canvas explicitly. Switch to svg at 50,000+ bars to feel the difference: one node per bar instead of a few dozen total.
