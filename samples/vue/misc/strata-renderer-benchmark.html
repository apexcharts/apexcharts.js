01Opt in, one line

Set chart.renderer: 'auto' and Strata switches to canvas once the mark count crosses rendererThreshold. Below it, output is identical SVG.

02Hybrid, not a rewrite

Only the series paths and markers move to canvas. Axes, gridlines, crosshairs, tooltips and annotations stay real SVG: crisp text, native events, SVG export.

03Tree shakeable

The interface and SVG adapter are core. The canvas backend is a separate feature, import 'apexcharts/features/renderer-canvas', that shakes out when unused.
